Markdown  
\# 履修科目選択ナビゲーター Webアプリ 要求仕様書

\#\# 1\. プロジェクト概要  
大学新入生（1年生）を対象とし、迷わずに卒業要件（100単位）に向けた履修計画を立てられるように導くスマートフォンファーストのSPA。  
アドバイザー教員と対話するような「診断フェーズ」を経て、学生個人の志向に合わせた「おすすめ科目」を提案。「シミュレーションフェーズ」で実際の科目をゲーム感覚で選択し、各種グラフで学習バランスを可視化する。

\#\# 2\. 技術スタック  
\* \*\*フロントエンド:\*\* HTML5, CSS3, Vanilla JavaScript (ES6+)  
\* \*\*グラフ描画ライブラリ:\*\* Chart.js (CDN経由)  
\* \*\*構成:\*\* サーバーレス静的サイト。バックエンドは持たず、データはJSのJSON配列として保持。

\#\# 3\. データ構造 (data.js)  
全科目データは、CSVの各DP小項目（DP1-1〜DP4-3の全20項目）を個別のパラメータとして保持する。
実データは同じディレクトリ内の”data.js”に記録し、これをデータベースに組み込む。

## **4\. アプリケーションのフローと機能要件**

### **4.1. 診断フェーズ / 4.2. 結果表示フェーズ**

4.1. 診断フェーズ (アドバイザーによる質問)
画面に親しみやすいアドバイザー（教員）のアイコンを表示し、チャット形式または1画面1問のステップ形式で以下の3問を提示する。

Q1. 興味のある分野は？（複数選択可）

音楽表現 / エンターテインメント / 舞台芸術 / 情報技術

Q2. 将来の関わり方・フィールドは？（複数選択可）

パフォーマンス（表に出る） / クリエイト（創り出す） / ビジネス（企画・運営する）

Q3. 特に伸ばしたい能力は？（3つまで選択可）

"1-1": "本質を見抜く力（審美眼）", "1-2": "創造力・発想力", "1-3": "設計・構築力", "1-4": "論理的思考力", "1-5": "自己理解・自己実現力", "1-6": "独創的な芸術表現力",
"2-1": "基礎的知識・技能の修得", "2-2": "標準的専門知識・技能の修得", "2-3": "発展的専門知識・技能の修得", "2-4": "専門知識・技能の応用・実践力", "2-5": "専門知識・技能の統合力","3-1": "文章読解力", "3-2": "文章表現力", "3-3": "協調性・協働する力", "3-4": "基礎的ITスキル", "3-5": "プレゼンテーション能力", "3-6": "異文化理解力・外国語能力",
"4-1": "共感力・相互理解力", "4-2": "規律性・倫理観", "4-3": "自己研鑽力（学び続ける力）"

ロジック: ユーザーの回答に基づいて、courseDataの各科目をスコアリングする。

Q1の選択肢と domains が一致すれば +2点

Q2の選択肢と fields が一致すれば +2点

Q3の選択肢に該当する dp の値（1または2）をそのまま加点

合計スコアの上位科目に isRecommended: true のフラグを付与する。

4.2. 結果表示フェーズ
「あなたには【〇〇分野】の【〇〇フィールド】がぴったり！ゼミの【〇〇先生】の領域です」と結果を表示する。

その先生が推奨する履修モデル（画像ファイル model_xxx.jpg またはPDFへのリンク）を表示する。

「さあ、シミュレーションを始めよう！」ボタンで次へ進む。

### **4.3. 履修シミュレーションフェーズ**

* **ダッシュボード (画面上部・固定推奨):**  
  * **プログレスバー:** 「現在〇〇 / 100単位」。単位追加でゲージが伸びるアニメーション。  
  * **DPチャート (Chart.js レーダーチャート):**  
    * **【重要】トグル切り替え機能:** 「4軸（大項目）」と「20軸（小項目）」を切り替えるトグルボタンをグラフ付近に配置する。  
    * デフォルトは「4軸（大項目の合算値）」を表示し、ユーザーがボタンを押すとChart.jsのデータを更新して「20軸」の表示へ切り替える（再描画させる）。  
  * **円グラフ (Chart.js):** 「分野」と「フィールド」の割合を2つの円グラフで描画。  
* **科目選択エリア (画面下部):**  
  * アコーディオンUIでカテゴリごとにリスト表示。各科目にチェックボックスを配置。  
  * `isRecommended: true` の科目には「🌟おすすめ」バッジを表示。  
* **インタラクション:**  
  * チェックボックスの操作に連動し、プログレスバーと現在表示中のグラフが即座に再計算・再描画されること。
